Esta é uma aplicação web de desenho simples que permite ao usuário desenhar ou apagar na tela usando uma ferramenta de pincel ou borracha. A aplicação é construída com JavaScript e HTML, utilizando o elemento <canvas> para renderizar o desenho.

Funcionalidades

Escolha entre uma ferramenta de pincel ou borracha.
Selecione o tamanho do pincel.
Escolha a cor do pincel.
Limpe a tela completamente.

Uso

Abra o arquivo HTML no seu navegador.
Selecione a ferramenta desejada (pincel ou borracha) clicando nos botões correspondentes.
Escolha o tamanho do pincel entre as opções disponíveis.
Escolha a cor do pincel usando a entrada de seleção de cor.
Comece a desenhar na tela clicando e arrastando o mouse.
Use a ferramenta de borracha para apagar áreas desenhadas.
Para limpar a tela, clique no botão "Limpar Tela".
